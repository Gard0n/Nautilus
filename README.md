# Renaissance digitale - Workshop

## Resume du brief
- Contexte: agence Product / e-commerce. Le client possede une base d'oeuvres du domaine public.
- Objectif business: ne pas vendre directement. Constituer un CRM ultra-qualifie pour futures campagnes de monetisation.
- Mission: concevoir une landing page e-commerce ou le produit est a 0 EUR en echange de donnees utilisateurs qualifiees (lead gen).
- Interdit: un simple telechargement (bouton PDF / image). Il faut une experience augmentee autour de l'oeuvre.
- Question cle: pourquoi donner ses donnees pour une oeuvre gratuite? La strategie UX et le concept font la difference.

## Contraintes
- AI first: IA pour contenus, visuels, voix, code.
- No-code automation: connecter des outils (IFTTT, Zapier, Make...).
- Mobile-first UX: design, flow, biais cognitifs.

## Livrables attendus
- Strategie AARRR
- Landing page (maquette Figma mobile-first, atomic design, copywriting persuasif, biais cognitifs)
- Data & mesure (3 a 5 KPIs SMART + methode de test)
